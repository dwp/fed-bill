Send a request for information

Use this form to send a Request for Information (RFI) to a provider under the Social Security Administration Act (SSAA) 1992, Section 190 B and C.

If you have already contacted a provider through other channels, you need to record your request.

You cannot use this form to request communications data under the Investigatory Powers Act (IPA) 2016. You will need to complete an IPA RFI instead. If you are not sure if your request falls under the IPA, email the Communications Data Unit IPA desk for more support: IPA.desk@dwp.gov.uk

Before you start

Before you send an RFI to a provider, you should have:

  • carried out all the relevant less intrusive checks
  • considered if your request is necessary and proportionate
  • considered if your request is lawful and complies with current legislation

What you will need

You will need to tell us:

  • what benefit your request relates to
  • what provider you want to send the request to, as well as their email or postal address if they are not known to us
  • the information you want to request
  • any information you already have about the request

We will use your answers to create a letter for the provider. This letter will explain your request and tell the provider how to respond.

You will need to:

  • check that the information in the letter is correct
  • confirm that we can send the letter to the provider
